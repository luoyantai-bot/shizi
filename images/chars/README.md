# 字卡图片目录

将每个汉字的象形图/记忆辅助图放在这个目录下。

## 命名规则

每张图片以**汉字本身**命名，格式为 PNG：

```
人.png
口.png
手.png
头.png
山.png
水.png
火.png
...
```

## 图片规格建议

- **格式**: PNG（推荐）或 WebP
- **尺寸**: 400×400px 左右即可（卡片上最大显示 200px 宽 × 140px 高）
- **背景**: 透明或白色
- **大小**: 每张控制在 50-200KB
- **风格**: 象形演变图、图解记忆、简笔画均可

## 如何使用

1. 将图片文件放入此目录
2. 无需修改任何代码，卡片翻面时会自动显示对应图片
3. 如果某个字没有对应图片，卡片背面会自动隐藏图片区域（不会报错）

## 完整字表

需要准备图片的所有汉字在 `src/data/characters.ts` 中定义，
当前共涵盖低/中/高三个难度约 1000 个汉字。
